Below is a step by step instruction showing how to validate the forms which uses the Skeleton framework.

Downloading the files

  • Create a file named skeleton.js with the following content:
{% highlight javascript %} {% include framework/skeleton/skeleton.js %} {% endhighlight %}
  • Create a file named skeleton.css with the following content:
{% highlight javascript %} {% include framework/skeleton/skeleton.css %} {% endhighlight %}

You should place skeleton.js and skeleton.css in formvalidation/dist/js/framework and formvalidation/dist/css/skeleton.css respectively.

Including the library

{% highlight html %} {% endhighlight %}

Calling the plugin

When calling .formValidation(), you need to set the framework option:

{% highlight javascript %} $(form).formValidation({ framework: 'skeleton', ... }); {% endhighlight %}
Some of forms use the row option to support multiple fields in the same row container
